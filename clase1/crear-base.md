# Crear base de datos en SQL
> para crear una base de datos vamos a utilizar el comando **CREATE DATABASE**

> En el panel de código escribimos: 

    CREATE DATABASE nombre;  

> Para este curso nos vamos a referir a una base de datos llamada "introdb"

    CREATE DATABASE introdb;  


> Ejemplo avanzado

    CREATE DATABASE nombre   
      CHARACTER SET utf8mb4  
      COLLATE utf8mb4_0900_ai_ci;  


## Eliminar una base de datos
> Para eliminar una base de datos vamos a utilizar el comando **DROP DATABASE**

> En la paleta de código escribimos:

    DROP DATABASE nombre;
